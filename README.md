Question:

          (Compute expressions) Write a program that displays the result of 9.5 * 4.5 - 2.5 * 3 / 45.5 - 3.5
          
Code:

    public class oneFive {
	    public static void main (String args[]){
		
		  double a = ((9.5 * 4.5) - (2.5 * 3))/(45.5 - 3.5);
		  System.out.println(a);
		
	    }
    }
